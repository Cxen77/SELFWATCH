# v1.5: Territorial Ownership & Dynamic Suppression Lifetime

Date: 2026-05-23

## Problem
In complex or long occlusions, identities could eventually switch because the standard suppression window was too brief. Spawning a new ID nearby would occur before the established track could recover, resulting in identity fragmentation and eventual switching.

## Key Solutions

### 1. Dynamic Suppression Lifetime Scaling
- **Trigger**: Instead of a fixed window, lost tracks calculate a dynamic suppression window from **45** up to **150** frames.
- **Scaling Formula**:
  - `base_window = 45`
  - `history_bonus = min(80, total_hits / 3)`
  - `recovery_bonus = min(25, recovery_count * 10)`
  - `confidence_bonus = 25` (if score >= 0.8)
- **Effect**: Highly stable tracks with rich histories guard their spatial regions and corridors significantly longer.

### 2. Territorial Ownership & Motion Corridor Gating
- **Concept**: A lost track owns a physical "territory" composed of:
  - **Spatial region**: Direct box overlap (IoU > 0.12).
  - **Trajectory corridor**: Predicted location along its velocity vector (`pred_cx = last_cx + vx * time_since_update`).
  - **Motion alignment**: Displacement direction matches track velocity direction (dot product > 0.3).
- **Effect**: Any new track spawning in this territory is treated as suspicious and suppressed by default.

### 3. Reversed Birth Assumption
- **Concept**: Instead of "new detection = new identity", any detection falling inside a lost/frozen track's territory is assumed to be the same returning person until proven otherwise.
- **Effect**: Suppresses spawning of competing new tracker IDs, preserving the slot for the historical identity to recover via appearance or CBIoU.

### 4. Historical Importance Weighting
- **Concept**: Long-lived stable identities require much stronger evidence to replace.
- **Effect**: If the current owner of an identity is historically stable (`total_hits > 50`), `GlobalIdentityManager` increases the required challenger inertia frames by `min(15, total_hits / 20)`.
