# v0.2 — StrongSORT Integration

**Date**: 2026-04 (approximate)
**Git Tag**: `v0.2.0` (retroactive)

## Motivation

Add appearance-based association to reduce ID switches. ByteTrack alone cannot distinguish between people with overlapping trajectories.

## What Changed

- Integrated OSNet (x1_0, MSMT17-pretrained) for appearance embeddings
- StrongSORT tracker with appearance + motion cost matrix
- Embedding gallery per track (rolling buffer of N best embeddings)
- Cosine distance gating for appearance matching

## Impact

| Metric | v0.1 | v0.2 | Delta |
|--------|------|------|-------|
| ID Switches | High | Moderate | Improved |
| FPS | ~25 | ~15 | -40% |

- `trackers/strongsort_tracker.py`
