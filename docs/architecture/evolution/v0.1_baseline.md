# v0.1 — ByteTrack Baseline

**Date**: 2026-04 (approximate)
**Git Tag**: `v0.1.0` (retroactive)

## Motivation

Establish a working multi-object tracking baseline with reliable detection and simple association.

## What Was Implemented

- YOLOv8 detector (nano → medium variants)
- ByteTrack association (IoU-based, dual threshold)
- Basic camera input pipeline
- Minimal UI overlay

## Key Metrics

| Metric | Value |
|--------|-------|
| FPS | ~25 |
| ID Switches | High |
| Identity Persistence | None (no memory) |

## Limitations

- No appearance matching — pure motion/IoU
- Identity lost immediately on occlusion
- No memory or re-identification capability
- Frequent ID switches in crowded scenes
