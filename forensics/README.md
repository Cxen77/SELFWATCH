# Forensic Failure Analysis

This directory stores:
- failure clips
- screenshots
- trajectory logs
- forensic metadata
- root-cause analysis

Current main failure categories:
- opposite-direction rebinding
- duplicate boxes
- pose-change fragmentation
- crowd overlap instability
- occlusion recovery failure

Each failure should contain:
- screenshot
- video clip
- explanation
- suspected cause
- fix attempt
- result after fix
